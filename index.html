$ man phosphor

    
PHOSPHOR(1)

    

      
NAME
phosphor — an operating system for people who miss the future that never shipped.

      
SYNOPSIS
A single-user environment where every window is a text window and the cursor never stops blinking. No notifications. No feed. Just you and the machine.

      
DESCRIPTION
Green light on black glass. A boot sequence you can read. Software that respects your attention because it barely has a UI to distract you with.
